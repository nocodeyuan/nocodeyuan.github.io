    选择器

    1. 基本选择器
      • 标签选择器:$("元素")
      • id选择器:$("#id")
      • 类选择器:$(".class")
      • 并集选择器：$("a,b")
    2. 层级选择器
      1. 后代选择器：$("A B"):选择A内部的所有B元素
      2. 子选择器：$("A > B"):选择A元素内部的所有B子元素
    3. 属性选择器
      1. 属性名称：$("A[属性名]")：获取拥有固定属性名的元素
      2. 属性：$("A[属性名='值']")：获取固定属性名值的元素
      3. 复合属性：$("A[属性名='值'][]...")：获取属性多条件的元素
      4. 非xx属性：$("A[属性名!='值']")：获取属性值不是xxx的元素
      5. 以xx属性开始：$("A[属性名^='xxx']")：获取属性值以xxx开始的元素
      6. 以xxx结束：$("A[属性名$='xxx']")：获取属性值以xxx结束的元素
      7. 包含xxx：$("A[属性名*='xxx']")：获取属性值包含xxx的元素
    4. 过滤选择器
      1. 首元素选择器
        • :first
      2. 尾元素选择器
        • :last
      3. 非元素选择器
        • :not(selector选择器)
      4. 偶数选择器
        • :even 偶数，从0开始计数
      5. 奇数选择器
        • :odd 奇数 ，从0开始计数
      6. 等于索引选择器
        • :eq(index) 指定索引元素
      7. 大于索引选择器
        • :gt(index) 大于指定索引元素
      8. 小于索引选择器
        • :lt(index) 小于指定元素
      9. 标题选择器
        • :header 获得标题（h1-h6）元素，固定写法
    5. 表单过滤选择器
      1. 可用元素选择器
        • :enable：获取可用元素
      2. 不可用元素选择器
        • :disable：不可用
      3. 选中后选择器
        • :checked 获取单选/复选框选中的元素
        • :selected 获取下拉框选中的元素

    DOM操作

    1. 内容操作
      1. html():获取/设置元素的标签体内容
      2. text():获取/设置元素的标签体纯文本内容
      3. val():获取/设置元素的value属性值
    2. 属性操作
      1. 通用属性操作
        • attr():获取/设置元素的属性
        • remoneAttr():删除元素属性
        • prop():获取/设置元素的属性
        • removeProp():删除元素属性
        • 区别：
          • 操作元素固有属性，使用prop
          • 操作自定义属性，使用attr
      2. 对class属性操作
        • getClass():获取class属性值
        • removeClass():删除class属性值
        • toggleClass():切换class属性值
          • toggleClass(“one”):判断元素是否存在class=“one”，存在删除，不存在添加
    3. CRUD操作
      1. append()：父元素将子元素追加到末尾
        • o1.append(o2):将o2添加o1末尾,父子关系
      2. prepend()：父元素将子元素追加到开头
      3. appendTo():
      4. prependTo():
      5. after():添加元素到元素后
        • o1.after(o2):将o2添加到o1后面，兄弟关系
      6. before():添加元素到元素前
      7. insertAfter()
      8. insertBefore()
      9. remove():移除元素
        • obj.remove()
      10. empty():清空元素的所有后代元素

动画

  1. 默认显示隐藏动画
    • show([speed,[easing],[fn]])
      • 参数
        • speed
          • 动画的速度。三个预定义的值（slow，normal，fast）或者动画时长毫秒值
        • easing：用来指定切换效果，默认’swing‘，可用参数‘linear’
          • swing：动画执行先慢，中间快，最后又慢
          • linear：动画匀速执行
        • fn：在动画完成时执行的和函数，每个元素执行一次
    • hide()
    • toggle()
  2. 淡入淡出动画
    • slidedown()
    • slideup()
    • slideToggle()
  3. 滑动显示隐藏
    • fadeIn()
    • fadeout()
    • fadeToggle()

AJAX

  1. 概念：ASynchronous JavaScript And Xml 异步的JavaScript和XML

  2. 同步与异步

    • 同步：客户端必须等待服务器端的响应。在等待的期间客户端不能做其他操作
    • 异步：客户端不需要等待服务器端的响应，在服务器处理请求的过程中，客户端可以进行其他操作

  3. 实现方式：AJAX是一种不需要刷新网页的情况下，能够更新部分网页的技术

JSON

  1. 概念：JavaScript Object Notation JavaScript对象表示法，现在多用于存储和交换文本信息的语法，进行数据传输
  2. 语法
    1. 基本规则
      • 数据在名称/值对中：json数据是由键值对构成
        • 键用引号包裹
        • 值的取值类型：
          1. 数字（整数/浮点数）
          2. 字符串（在双引号中）
          3. 逻辑值（true/false）
          4. 数组（在方括号中）
          5. 对象（在花括号中）
          6. null
      • 数据用逗号分隔：多个键值对由逗号分隔
      • 花括号保存对象
      • 方括号保存数组
    2. 获取数据
      1. json对象 .键名
      2. json对象.[“键名”]
      3. 数组对象[索引]
    3. JSON解析器
      1. Jsonlib
      2. Gson
      3. fastjson
      4. jackson！！
    4. JSON数据和Java对象的相互转换
      1. JSON转为Java对象
        1. 导入Jackson相关jar包
        2. 创建Jackson核心对象ObjectMapper
        3. 调用方法
          • readValue(json字符串数据, Class)
      2. Java对象转为JSON
        1. 导入Jackson相关jar包
        2. 创建Jackson核心对象ObjectMapper
        3. 调用方法
          • writeValue(参数1，obj)：
            • 参数1：
              • File：将obj对象转换为JSON字符串，并保存到指定文件中
              • Writer：将obj对象转换为JSON字符串，并将JSON数据填充到字符输入流
              • OutputStream
          • writeValueAsString(obj)：将对象转为JSON字符串
        4. 注解：
          1. @JsonIgnore：排除属性：转化后不显示该属性
          2. @JsonFormat：定义在属性声明或者get方法上，属性值的格式化：@JsonFormat（pattern = “xxx”）转换为固定xxx格式
        5. 复杂的Java对象转换
          1. List：转换后为数组包裹的json字符串
          2. Map：转换后格式与对象转换一样
      3. 校验用户名是否存在
        1. 服务器响应的数据，在客户端使用时，要想当作json数据格式使用
          1. $get(type)：将最后一个参数type指定为“json”
          2. 在服务器端设置响应数据的MIME类型，application/json
